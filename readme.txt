== Description ==

Self hosted privacy friendly analytics plugin with real time visitors counter and super advanced analytics dashboard with the most important statistics like pageviews, sessions, bounce rate, unique visitors, top pages, top referrers, etc.,

https://youtu.be/XdpizPvNKjE

As the name states, Cookieless Analytics don't use ANY cookies to measure analytics. Enjoy privacy friendly analytics right in your WordPress admin dashboard.

== Installation ==	
1. Make a backup of your website and database
2. Download the plugin
3. Upload the plugin to the wp-content/plugins directory,
4. Go to “plugins” in your WordPress admin, then click activate.
5. You will now see Cookieless Analytics page right below dashboard in left sidebar menu.

== Frequently Asked Questions ==

= How long it needs to start showing analytics after plugin installation? =
Measurement of statistics will be started right after plugin activation and you can see top pages, top referrers soon after few minutes.

= Does it impact on site performance? =
Cookieless Analytics uses REST API to log statistics in background asynchronously without impacting frontend site performance.

= Does search engine bot traffic reflect in analytics? =
All major search engine bots traffic are safely ignored and will not be measured in analytics.

= Is logged in user session measure in analytics? =
Yes, it is measured as of now.
